# 全局深色主题设计方案

## 概述
将标准物质管理系统从浅色主题改造为现代深色主题，参考设计风格包含玻璃态效果、渐变图标、微动画等视觉效果。

## 设计目标
- 全局深色主题覆盖
- 深蓝灰底色 + 紫蓝渐变强调色
- 玻璃态卡片效果
- 流畅的过渡动画

## 配色系统

### 背景层次
```css
--bg-primary: #0f172a;      /* 主背景 */
--bg-secondary: #1e293b;    /* 卡片底色 */
--bg-tertiary: #334155;     /* hover/激活态 */
```

### 文字颜色
```css
--text-primary: #f1f5f9;
--text-secondary: #94a3b8;
--text-muted: #64748b;
```

### 强调色
```css
--accent-purple: #8b5cf6;
--accent-blue: #3b82f6;
--accent-cyan: #06b6d4;
--accent-pink: #ec4899;
```

### 玻璃态效果
```css
--glass-bg: rgba(30, 41, 59, 0.7);
--glass-border: rgba(148, 163, 184, 0.1);
--glass-shadow: 0 8px 32px rgba(0, 0, 0, 0.3);
```

## 组件样式

### 统计卡片
- 左侧 4px 渐变边框条
- 渐变背景图标
- 玻璃态背景
- hover 上浮 + 发光效果

### 普通卡片
- 16px 圆角
- 玻璃态背景
- 微妙边框

### 数据表格
- 深色表头
- hover 行高亮
- 彩色状态指示器

### 按钮
- 主要：紫蓝渐变 + hover 发光
- 次要：透明边框 + hover 渐变
- 危险：红色渐变

## 动效系统

### 卡片悬浮
- translateY(-4px)
- 紫色光晕阴影
- 边框发光

### 渐变动画
- 图标背景渐变流动
- 3s 循环

### 菜单过渡
- hover 背景变化
- 左侧边框指示

### 页面入场
- fade-in-up 动画
- 错开延迟

## 实现方案
采用 CSS 变量 + Element Plus 暗黑模式覆盖方案：
1. 创建全局 CSS 变量文件
2. 覆盖 Element Plus 暗黑模式变量
3. 添加玻璃态、渐变等自定义效果
4. 改造布局和组件样式
