# 标准物质管理系统 - 需求规格说明书

## 1. 项目概述

### 1.1 项目背景
实验室标准物质管理是实验室质量管理体系的重要组成部分。目前使用Excel进行管理存在以下问题：
- 按储存温度分Sheet，查询不便
- 字段命名不统一
- 缺少完整的入库/出库追溯记录
- 有效期管理靠人工
- 无库存预警机制

### 1.2 项目目标
构建一个符合 ISO 17025 标准的实验室标准物质管理系统，实现：
- 标准物质全生命周期管理
- 出入库审批流程
- 自动预警机制
- 完整审计追踪

### 4.2 基础数据模块

#### 4.2.1 标准物质主数据
- 维护标准物质的基础信息（编号、名称、标准值、规格等）
- 独立于库存，入库时选择关联
- 支持按分类筛选查询

#### 4.2.2 分类管理
- 树形分类结构
- 如：对照品、对照药材、标准溶液、滴定液等

#### 4.2.3 位置管理
- 管理存放位置（货架、冰箱、柜子等）
- 记录位置的温度要求、容量等信息

#### 4.2.4 元数据配置
- 入库原因：新购入、盘盈、归还、调拨入、其他
- 出库原因：实验使用、过期销毁、报废、调拨出、赠送、其他
- 储存条件：-20℃、2-8℃、常温、阴凉干燥、避光等
- 审批状态：待审批、已通过、已拒绝、已撤回

### 4.3 库存管理模块

#### 4.3.1 库存查询
- 多条件筛选（分类、名称、批次、有效期等）
- 列表展示库存明细
- 支持导出Excel

#### 4.3.2 预警中心
三种预警类型：
1. **有效期预警**：提前1-3个月提醒将要过期的标准物质
2. **库存不足预警**：库存低于设定值时提醒补货
3. **长期未使用预警**：入库后X个月未使用提醒

### 4.4 出入库管理模块

#### 4.4.1 入库登记
流程：选择标准物质 → 填写批次信息 → 指定存放位置 → 提交入库

#### 4.4.2 出库申请与审核
流程：
1. 普通用户提交出库申请
2. 标准物质管理员审核
3. 审核通过后完成出库

#### 4.4.3 出入库记录
- 完整的出入库历史记录
- 支持按时间、操作人、物质等筛选

### 4.5 采购管理模块

#### 4.5.1 供应商管理
- 供应商信息维护（名称、联系人、电话、地址）

#### 4.5.2 采购流程
1. 提交采购申请
2. 管理员审核
3. 到货验收
4. 验收通过后自动入库

### 4.6 盘点管理模块

#### 4.6.1 盘点任务
- 创建定期盘点任务
- 指定盘点范围（按分类、位置等）

#### 4.6.2 盘点执行
- 逐项确认库存
- 记录实盘数量

#### 4.6.3 差异处理
- 显示账实差异
- 处理盘盈盘亏

### 4.7 系统管理模块

- 用户管理：账号维护、密码重置
- 角色权限：角色配置、权限分配
- 操作日志：系统操作审计追踪

## 5. 数据库设计

### 5.1 核心表结构

| 表名 | 说明 | 关键字段 |
|------|------|----------|
| `reference_material` | 标准物质主数据 | id, code, name, standard_value, specification, category_id |
| `category` | 分类管理 | id, name, parent_id, sort_order |
| `location` | 存放位置 | id, code, name, temperature, capacity |
| `stock` | 库存记录 | id, material_id, batch_no, internal_code, expiry_date, quantity, location_id |
| `stock_in` | 入库单 | id, stock_id, reason, quantity, supplier_id, operator_id, create_time |
| `stock_out` | 出库单 | id, stock_id, reason, quantity, applicant_id, approver_id, status, create_time |
| `purchase` | 采购单 | id, material_id, quantity, supplier_id, status, applicant_id, approver_id |
| `stock_check` | 盘点单 | id, check_date, scope, status, create_by |
| `stock_check_item` | 盘点明细 | id, check_id, stock_id, system_qty, actual_qty, difference |
| `supplier` | 供应商 | id, name, contact, phone, address |
| `metadata` | 元数据配置 | id, type, code, name, sort_order |
| `user` | 用户 | id, username, password, role_id, status |
| `role` | 角色 | id, name, permissions |
| `operation_log` | 操作日志 | id, user_id, action, target, detail, create_time |

### 5.2 表关系说明

```
category (1) ────── (N) reference_material
                           │
                           ▼
location (1) ────── (N) stock (N) ────── (1) reference_material
                           │
              ┌────────────┼────────────┐
              ▼            ▼            ▼
         stock_in    stock_out    stock_check_item
              │            │            │
              ▼            ▼            ▼
         supplier    (审批流程)    stock_check
```

## 7. 数据迁移

### 7.1 现有数据结构分析
现有Excel包含10个Sheet，按储存温度分类：
- 化妆品扩项标准物质（-20℃、2-8℃、常温）
- 化妆品中金属元素
- 滴定液（常温、阴凉处）
- 对照药材（10-30℃）
- 对照品（10-30℃、2-8℃、-20℃）

### 7.2 数据清洗与导入
1. 统一字段命名
2. 按新分类重新整理
3. 创建位置数据
4. 导入标准物质主数据
5. 导入库存记录
